Crystal Collector!

A random number will be generated at the start of the game.

There are four crystals below. Each crystal has a random value set to a specific amount of points. Clicking each crystal will add points to your total score.

Win the game by matching your total score to the random number generated. Be careful, if your total points goes above the generated number, you lose.

Each time a new game starts, the value of the crystals and random number will be changed.
